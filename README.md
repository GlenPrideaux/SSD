# SSD
Samuel, Saul and David parallel edition

## Build
To build, you require
- XeLaTeX and a number of associated packages
- pandoc
- TeX Gyre Pagella font
- Ezra SIL font
- Python3

Pretty much everything else should be present on a typical Mac or GNU/Linux box. (I don't know about Windows ... you'll need corresponding resources but I imagine there will be some tweaking required to get it to build.)
